# I.B.U. Dashboard

Intelligent Beings United

## What is it?

This is a project made by members of the [I.B.U. Sheepit team](https://sheepit-renderfarm.com/team/2109), designed to track user contribution to the team. You can see the production deployment [here](https://ibu.koolkid6958.dev).

## Dev Quickstart

> [!NOTE]
> This project is designed to run on GNU/Linux, however you should be able to develop it on Windows and macOS.

### Requirements

- Rust
- Python 3

### Instructions

- Clone this repo.
- Run `./scripts/setup.sh` To make a virtual python environment, build rust libraries, and install python dependencies.
- Run `./scripts/run-dev.sh` to run. You can also run manually with `./.venv/bin/python main.py`

If you want to run a production version refer to the [Production Setup Guide](docs/prod-setup.md)
